# HuguitisNodesBot
The simple bot used for the official Huguitis Nodes server. 

# Contributing:
Make a pull request and we will check over it!

# Setup:
1. Fill Out The Config In Settings.js
2. Run `npm i`
3. Run `node main.js`
4. Done! If you want to change more then that is fine!
